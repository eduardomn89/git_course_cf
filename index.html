git init: inicializa la carpeta para trabajar con git. al dar el comando se indica que se esta trabajando en el working directory, para despues pasarlos archivos a staying area y finalmente al repositorio.

git status: para ver que archivos estamos trabajando.

git add: agregar archivos al area de trabajo.Los que no estan en el area de trabajo o no son seguidos estan en rojo

Git commit: es como una foto de lo que se tiene hasta el momento.
